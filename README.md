## ✨ Features

This portfolio is built entirely from scratch without relying on heavy frontend frameworks, focusing on pure web fundamentals and custom JavaScript functionality:

*   **🌗 Dark / Light Mode:** Fully implemented theme toggle using CSS variables for a seamless reading experience in any environment.
*   **🌐 Bilingual Support (TH / EN):** Real-time language switching without page reloading using custom JavaScript class manipulation.
*   **✉️ Functional Contact Form:** Integrated with **Formspree** via Fetch API. Submissions are handled asynchronously (AJAX) with custom success/error states, avoiding disruptive page redirects.
*   **🎬 Dynamic Animations:** Features a custom typewriter effect in the hero section and scroll-reveal animations that trigger as elements enter the viewport.
*   **📄 Resume Modal:** A built-in modal viewer for the resume image, including a click-to-zoom feature.
*   **📱 Fully Responsive:** Optimized layout for desktop, tablet, and mobile devices, including a responsive bottom navigation bar for mobile screens.
